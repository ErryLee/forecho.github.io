关于本博客的使用

这个博客不是我的一个博客，之前用了3年的 WordPress 了，这个博客也刚开始用，目前只是为了体验以 Markdown 写作为主。
那么前几天也试过用 Hexo 的插件把我之前博客的文章有导入，但是后来又把导入的文章全部删掉了。想着借这次机会从头来过。

那么这个博客用来写什么呢？

如你所见，这个博客分两大类 - 技术栈和人文栈。
身为程序员技术的成长当然是必不可少的，但是同时也要看去看一下非技术类的书，去思考心理学、哲学，这将会是一件很有趣的事情。
除了这些外，我还会写一个读书笔记。

博客的说明

博客不会再需要其他分类了，所有的文章主要靠标签来区分管理。所有的文章都会是原创。
用着 Markdown 的好处就是回归写作的初心，我们只需要关注写作本身就可以了。

好了，我说完了，以上皆是废话。
