我的 Chrome 扩展集

遇见 Chrome 浏览器

在大一的时候才开始拥有一台属于自己的电脑，那个时候买的还是二手台式机。之前的我可以说算是电脑小白了，根本就没有浏览器这个概念，就跟现在绝大多数的中国网民一样。只知道有个图标你点一下就能上网，后来才知道它就是 IE 浏览器。

也不知道什么时候开始什么原因，让我知道了 Chrome 浏览器，初次使用的时候，非常的不习惯。不习惯以下几点：

  1. 以标签形式展现一个网页。
  2. 头部很短。
  3. 界面极简。（但功能强大）

其实用了你才发现这些都是它的优点，再加上它的一个帐号多平台同步功能和扩展机制，我再也离不开它了。可以说 Chrome 浏览器把 「Don’t Make Me Think」发挥的淋漓尽致，这一点都不为过。

国内的什么猎豹浏览器，什么360极速浏览器都是在用 Chromium （Chrome 浏览器背后的引擎）基础上加一个「外壳」而已，但是就是用不习惯，用户体验太差。

「普遍」的扩展

这里是指使用率很高的扩展。

AdBlock

这个是用来屏蔽广告的，必备扩展，妈妈再也不用担心突然进入某个页面突然广告乱飞的情况了。再也不用担心使用百度时看到恶心的 flash 和大图广告了。

PS：有一个值得注意的情况：如果你是一个开发者，也就是工程师在写代码开发页面的时候，有可能启用这个扩展导致一些因屏蔽广告而导致的一些错位什么的问题，留一个心眼就好了。

Proxy SwitchySharp

翻墙必备扩展，翻墙教程我这儿有一个最简单的方法，屡试不爽。

豆藤 Bean vine

如果你经常使用豆瓣的话，用了这个扩展之后你就会停不下了的。

购物党

买东西的时候可以商品查看历史价格，非常实用。同类扩展还有惠惠网购助手。

印象笔记·剪藏

印象笔记，一款非常强大的多平台笔记本，最强大的功能，也就是我最喜欢的功能就是这个。如果我遇到一个问题，然后通过 Google 搜索找到解决办法，那么我会把这篇文章剪藏到我的印象笔记里面，非常实用，以防下次查阅。

印象笔记·悦读

看文章的时候，可能会受到旁边图片的干扰，点一下这个台灯图标，会重新排版内容，非常干净，舒服，那么现在你就可以安心的看文章了吧。

Readability

稍后阅读功能，有时候一篇文章在电脑上看到而当时有不适合阅读的时候，那么此款扩展正式你需要的，支持多平台，一键把文章保存到云端。

当初使用它，是因为刚买了一个 Kindle，而这个可以一键把文章发送到 Kindle。而现在我使用它的原因就因为我爱上它的 UI 了。

Save to Pocket

又一款稍后阅读软件，我拿来备用的。功能可能比 Readability 稍强一点，但是 UI 没有 Readability 好。

TabCloud

保存标签功能，支持 Google 帐号登录，也是非常实用。

想象一下你现在正在找资料解决问题，然后突然有事要关机或者换电脑了，那么有了这个扩展，你就不用担心你现在打开的页面丢失了。

优酷去广告 扩展

看视频去广告，你懂的。配置说明很简单。备用扩展Adkill and Media Download(此扩展可以下载视频)。

百度搜索

至于为什么有这个坑爹的扩展我要解释下，因为 Chrome 浏览器默认的搜索设置成 Google 了，虽然翻墙了，但是极少数情况下还是访问慢，你懂的。

用了这个扩展你就可以选中页面的某个文字，然后直接右键「百度一下」直接搜索，省掉了「复制文字」、「新建标签」、「输入百度网址」、「粘贴文字」搜索，真是懒人必备。

爱库网快速收藏工具

对于比较喜欢收藏书签的人，这绝壁是个好扩展。

Google Mail Checker

收 Google Mail 邮件的。

HTML5高清二维码生成器

有时候一个网页需要在手机端访问，现在支持二维码功能的 App 那么多，再加上这个扩展，这件事情就变得再简单不过了。

快捷翻译(中英文)

双击某个单词，自动翻译，非常方便。

开发者扩展

开发者扩展是针对我自己平常开发使用的扩展。

JSONView

当页面输出纯 JSON 数据的时候，会自动帮你处理数据显示的格式，嗯，很方便。

PHP Ninja Manual

手册用经常查才能熟，遇见不会的，疑惑的赶紧查。

Postman - REST Client

可以模拟 POST 请求，测试的时候很方便。

smartF5

有了这个扩展之后你修改过的 HTML、CSS，JavaScript 文件它都有监听到，自动刷新页面。懒人必备，妈妈再也不用担心我的 F5 键了。

PS：仅支持 file:/// 开头的路径。

Live Reload

腾讯出品。功能跟上面的差不多，但是支持本地搭建的服务器环境访问的页面。但是暂不支持通过 file:// 打开的页面，此扩展和上面的 smartF5 扩展配置使用，非常不错。
