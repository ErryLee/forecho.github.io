进阶高级 PHP

文章目录
  1. 1. 书籍
  2. 2. 网站
  3. 3. 开源项目
  4. 4. 框架

使用 PHP 也快3年之久了，对之「恨铁不成钢」。PHP 的优点又很多，如能快速建站，入门简单，框架一堆，开源项目一堆，市场需求很大，不愁找不到工作。但同样也是槽点很多，入门很简单但要深入学习却需要懂的很多，由于初期没有好的语言规范导致写出来的代码能运行，但是很不规范，导致各种怪异的代码都有，团队合作开发的时候，你就知道痛苦了。PHP 是一个脚本语言，但是如果想学好PHP，你必须要有很强的面向对象思维。目前自我感觉还是处于中级 PHPer，对于很多功能实现是没问题的，但是对其原理知之甚少，也没有好好看过一个优秀的开源项目。故作此文，希望多看一些开源代码，多借鉴，多学习，多做笔记。

他山之石，可以攻玉。
